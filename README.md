# quest

- このレポジトリーはVRゲームの開発用です。  
- 自由にcloneやzipを落としてもらっても構いません。  
## 開発環境
- OS => Windows 10 
- CPU => Intel(R) Core(TM) i7-6700 CPU @ 3.40GHz
- RAM => 16.0GB
- UnityVersion => Unity 2018.1.6f1 (64-bit)
